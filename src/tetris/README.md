# Tetris in RATR0

## Description

This is a version of Tetris that was written for AmigaGameJam 2024. It is inspired by
Tetris 99 and Tetris Effect.
As such it implements modern Tetris rules and otherwise focuses on fluent gameplay


## Blitting strategy

Every block should take a maximum of 2 blits for displaying it and a maximum of 2
blits for deleting
